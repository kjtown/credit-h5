这个项目是基于[vue-cli3]创建的.
## vue-cli指令
  - npm install -g @vue/cli //全局安装vue-cli
  - vue create projectName  //创建项目
  - vue -h(vue --help)      //查看vue帮助
  - vue -V(vue --version)   //vue版本

## 项目指令
  - npm install    //安装本地依赖
  - npm run serve  //开发环境
  - npm run build  //生产环境
  - npm i axios -S //axios依赖
